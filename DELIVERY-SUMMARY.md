## 🎨 Customization Guide

### Change Primary Color
Edit `css/modern-style.css`:
```css
:root {
  --primary: #your-color;      /* Replace #6366f1 */
  --primary-dark: #your-color;   /* Replace #4f46e5 */
  --primary-light: #your-color;  /* Replace #818cf8 */
  --primary-glow: rgba(your-rgb, 0.4);
}
```

### Update Phone Number
Edit `js/modern-main-full.js`:
```javascript
footer_phone: "+212 YOUR_NUMBER",
```

### Update Business Hours
Edit HTML files directly or add to translation dictionary.

### Add New Language
1. Add translation object to `js/modern-main-full.js`
2. Add language button to HTML:
```html
<button class="lang-option" data-lang="es">Español (ES)</button>
```
