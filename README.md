# Electro Youssef - Phone Repair Website

Professional phone repair website for Electro Youssef (Youssef GSM) located in ZERARDA, Algeria.

## 📋 Overview

A modern, responsive, and multilingual phone repair website with:
- **5 Pages**: Home, Services, About, Pricing, Contact
- **3 Languages**: English (EN), French (FR), Arabic (AR) with full RTL support
- **Modern Design**: Glassmorphism, dark theme, smooth animations
- **WhatsApp Integration**: Direct booking via WhatsApp
- **DZD Pricing**: Transparent pricing in Algerian Dinar
- **Mobile Responsive**: Fully responsive design for all devices

## 🔧 Setup Instructions

### 1. Open the Website
Simply open `index.html` in a web browser. No server required.

### 2. Deploy to Hosting
Upload all files to any static hosting:
- GitHub Pages
- Netlify
- Vercel
- Any web hosting

### 3. Customize
Edit the following files:
- **HTML**: Update content in HTML files
- **CSS**: Modify colors and styles in `css/modern-style.css`
- **JS**: Update translations in `js/modern-main.js`
- **Images**: Add images to `images/` directory
